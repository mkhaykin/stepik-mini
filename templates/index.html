Описание: эта игра представляет собой пошаговую бродилку.
Цель игры: выбраться из лабиринта. Для забавы в игру может быть добавлен охотник (он же ниндзя). Не дайте ему приблизиться к Вам.
Управление: интуитивное ;). Стрелки вправо/влево, назад/вперед и пропуск хода. Важно:При первой генерации лабиринта вероятно некоторое "подтупливание" системы. Это связано с особенностями развертывания приложения на heroku.com.

Примечание:

  1. Чтобы уменьшить код, генерация лабиринта вынесена из проекта. Используется запрос к другому моему проекту: http://labyrinths.herokuapp.com/. Лабиринт генерируется случайно и, в определенных случаях, не может быть сгенерирован корректно. В этом случае, стоит начать игру заново и не снижать оценку ;).
  2. Динамический маршрут "дергается" из JS. Так же через него реализована перерисовка рабочего поля. Очень надеюсь, что использование JS не будет воспринято Вами как недочет.
  3. Для идентификации сессии используется cookies. Система не тестировалась на браузерах с запретом на использование cookies. Надеюсь, это не будет заявлено как ошибка.
  4. Также используются всплывающие окна. Если они заблокированы, поведение системы может отличаться от задуманного (все же это лабораторный проект).
  5. В проекте присутствуют дополнительные файлы: runtime.txt и procfile. Это не ошибка, файлы нужны для корректной публикации на heroku.com
  6. За иконки спасибо сайту: icons8.ru
